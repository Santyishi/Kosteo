# Kosteo

Aplicación web de gestión de gastos personales con registro manual y por WhatsApp.

## Stack
- **Frontend:** Angular + Angular Material
- **Backend:** NestJS + Prisma
- **Base de datos:** PostgreSQL
- **Auth:** JWT
- **WhatsApp:** Meta Cloud API (post-MVP)
